# 初中英语短语/固定搭配：校对 + 优化版

> 说明：本版对原文件全部条目做了**释义纠错/补充、词性(结构)、常见语境**与**例句**；并统一排版为 Markdown 表格。

## 全局勘误（原稿中需要更正的关键点）

- **in all**：原写“总之”不准确；常用义为 **“总共；一共”**。

- **come up with**：原写含“追上/赶上”不准确；应为 **“想出/提出（主意、办法、答案）”**。

- **as well ass**：拼写错误，应为 **as well as**。

- **compare with / compare to**：
  - compare **with**：比较异同
  - compare **to**：把…比作…/类比
